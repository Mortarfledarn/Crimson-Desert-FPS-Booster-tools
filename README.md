⚙️ FPS Boost Tool for Crimson Desert 🚀
A lightweight system optimization tool for Crimson Desert. Allows you to adjust CPU priority 🖥️, clear temporary files 🧹, and apply graphics tweaks 🎨. All changes are reversible 🔄.

🧩 How It Works
The tool does not modify game files 📁 — it optimizes how system resources are allocated ⚡. Just launch the file ▶️, select your preferred options ✅, and apply the settings — instantly and hassle-free 💪.

📖 Instructions
1️⃣ Download the latest version of the tool from the Releases section 📥.

2️⃣ Run the CrimsonDesert_FPS_Boost.exe file as administrator 👑 (right-click → "Run as administrator").

3️⃣ In the window that opens, select the desired options ☑️:

🎯 High CPU Priority — allocates more processor resources to the game.

🧹 Clear Temporary Files — frees up RAM.

🎨 Graphics Tweaks — additional optional optimizations.

4️⃣ Click the "Apply" button and wait for the process to complete 🔧.

5️⃣ Launch Crimson Desert and enjoy the game 🎮✨.

6️⃣ To revert to default settings, run the tool again and click "Restore" 🔄.

📌 Important Notes
🔒 The tool works only on Windows 10/11 (64-bit).

🚫 Before launching the game, close your browser and other heavy applications.

🧠 Only use the options you're familiar with to ensure maximum stability.

This project is intended for optimization and educational purposes 🛠️🎯
